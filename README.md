# Zenith Vault (Vite + React)

Multi-page Vite + React starter for Zenith Vault.

## Setup

1. Install dependencies:
```
npm install
```

2. Run dev server:
```
npm run dev
```

3. Build:
```
npm run build
```

4. Deploy (one-time using gh-pages):
```
npm run deploy
```

Or use the included GitHub Actions workflow which deploys on push to `main`.
